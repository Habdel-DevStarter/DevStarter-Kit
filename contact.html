Réponses rapides

Votre question est peut-être déjà répondue ici.

Les templates sont-ils vraiment gratuits ? ▼

Oui, totalement et définitivement. Tous les templates DevStarter Kit sont gratuits pour un usage personnel et commercial, sans limite de projets. Pas d'abonnement, pas de crédits requis, pas de « freemium ». Vous copiez, vous utilisez, c'est tout.

Puis-je utiliser les templates dans un projet client ? ▼

Absolument. Vous pouvez utiliser n'importe quel template dans vos projets personnels, vos projets clients, vos projets commerciaux, et les modifier sans restriction. La seule chose que j'apprécie (mais n'exige pas) : mentionner DevStarter Kit dans votre projet si vous en avez l'occasion.

Un template ne s'affiche pas correctement, que faire ? ▼

Vérifiez d'abord que vous avez bien copié le HTML et le CSS ensemble. Chaque template nécessite les deux. Si le problème persiste, signalez-le via ce formulaire en précisant votre navigateur, la version de l'OS et une description du comportement attendu vs observé. Je corrige généralement les bugs sous 48h.

Pouvez-vous créer un template sur mesure pour mon projet ? ▼

Le projet est entièrement bénévole et je ne propose pas de prestations sur mesure. En revanche, si votre besoin est suffisamment universel pour être utile à d'autres développeurs, je peux l'intégrer à la bibliothèque. Écrivez-moi avec une description détaillée et j'étudierai la demande.
