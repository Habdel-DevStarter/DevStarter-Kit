Défis débutants

10 mini-exercices progressifs pour pratiquer HTML, CSS et JavaScript en conditions réelles.

Défi 1 - Navbar sticky

Rendez la navbar fixe en haut avec un fond translucide.

.site-header{position:sticky;top:0;backdrop-filter:blur(8px);}

Défi 2 - Card hover

Ajoutez un effet lift au survol d'une card.

.card:hover{transform:translateY(-6px);box-shadow:0 14px 28px rgba(0,0,0,.12);}

Défi 3 - Grille responsive

Passez une grille 3 colonnes en 1 colonne sur mobile.

@media (max-width:768px){.grid--3{grid-template-columns:1fr;}}

Défi 4 - Dark mode

Stockez le thème choisi dans localStorage.

localStorage.setItem('devstarter-theme','dark');

Défi 5 - Copy code

Copiez le contenu d’un bloc code au clic.

navigator.clipboard.writeText(code.textContent);

Défi 6 - Validation email

Affichez un message d’erreur si email invalide.

if(!email.checkValidity()){message.textContent='Email invalide';}

Défi 7 - Menu mobile

Ouvrez/fermez le menu hamburger avec aria-expanded.

menu.classList.toggle('is-open');btn.setAttribute('aria-expanded', String(open));

Défi 8 - Filtre de recherche

Masquez les cards qui ne matchent pas la recherche.

item.style.display = matches ? '' : 'none';

Défi 9 - Toast UX

Affichez “Copié” 2 secondes après une action.

toast.classList.add('is-visible');setTimeout(()=>toast.classList.remove('is-visible'),2000);

Défi 10 - Scroll to top

Affichez le bouton seulement après 300px de scroll.

btn.classList.toggle('is-visible', window.scrollY > 300);
